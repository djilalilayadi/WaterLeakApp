## 3. Storage Permissions (CRITICAL)
If you get a "Row-level security policy" error when uploading photos, run this SQL to allow uploads to the `image` bucket:

```sql
-- Allow authenticated users to upload files to the 'image' bucket
CREATE POLICY "Allow authenticated uploads" ON storage.objects
FOR INSERT TO authenticated
WITH CHECK (bucket_id = 'image');

-- Allow anyone to view files in the 'image' bucket (since it's public)
CREATE POLICY "Allow public viewing" ON storage.objects
FOR SELECT TO public
USING (bucket_id = 'image');
```

## 3. How to Test
1. **Login as Technician:** Use your tech account. You should see the "Available Jobs" list sorted by distance.
